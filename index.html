eOntziApp es una aplicación web creada por alumnos del ciclo de desarrollo de aplicaciones web del instituto Zubiri Manteo.

Historia del proyecto eOntzia

El proyecto eOntzia, nació de la colaboración con los alumnos de electrónica de otros centros de Gipuzkoa, Don Bosco de Errenteria y La Salle Berrozpe de Andoain. Consiste en la creación, por parte de los alumnos de electrónica, de un dispositivo compuesto por diversos sensores controlados por una placa Arduino que iría colocado de tal manera que pudiera realizar mediciones del volumen de llenado de un contenedor de basura. Los datos son enviados vía Internet a una base de datos central, además de poder visualizar y gestionar dichos datos, la aplicación web generaría rutas optimizadas para la recogida de los residuos, con el objetivo de mejorar la recogida y reducir gastos. Tecnología utilizada en la aplicación

La aplicación está creada con HTML5, CSS3, JavaScript/JQuery y Bootstrap en la parte de cliente o front-end y PHP utilizando el framework Slim Framework en la parte de servidor o back-end, implementando una base de datos relacional MySQL.
